Acerca del Proyecto

Este proyecto es una plataforma de publicaciones y comentarios, donde los usuarios pueden registrarse, iniciar sesión, crear publicaciones, comentar, dar "me gusta" y gestionar su perfil.

Tecnologías utilizadas:

  • Django (backend y manejo de datos)
  • HTML y Bootstrap (interfaz y estilos)
  • JavaScript y AJAX (para interacciones dinámicas)

El objetivo es ofrecer una experiencia similar a una red social, donde la comunidad pueda compartir ideas y comunicarse de forma sencilla.
